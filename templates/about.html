About Fina

Smart Personal Financial Management by Leveraging Machine Learning and the Open Banking API

1. Introduction

1.1 Motivation

The development of Fina was fostered by two major factors in the current financial landscape:

State of Software Products

Modern software delivery has evolved significantly, with rapid updates and patches becoming the norm. However, for financial applications, we believe in maintaining the highest standards of quality and security from the start. Fina is built with a focus on reliability and optimization, ensuring a stable and secure experience for our users.

State of Personal Finances in Brazil

With the average Brazilian maintaining 5.2 bank accounts and the rapid rise of digital banking, there's a growing need for comprehensive financial management tools. Fina addresses this complexity by providing a unified platform for managing multiple accounts while promoting financial literacy.

2. Implementation

2.1 Architecture

Fina is built on a modern, secure architecture:

  • Server: Written in async Rust with HTMX for the web interface
  • Database: PostgreSQL with async SQLx for optimal performance
  • Mobile: Native Android application written in Kotlin
  • Machine Learning: Custom categorization model for transaction analysis

2.2 Security Measures

  • Two-factor authentication (2FA)
  • Read-only access to banking information
  • Regular security updates and monitoring

2.3 Core Features

  • Multi-bank account integration via Open Banking API
  • Intelligent transaction categorization
  • Financial analytics and insights
  • Manual transaction tracking for cash expenses
  • Interactive dashboards and reports

3. Open Source Commitment

Fina is an open-source project, available at github.com/finnish-app/. We believe in transparency and community-driven development, allowing developers to contribute to our mission of improving financial management for everyone.
